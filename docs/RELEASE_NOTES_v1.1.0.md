# GT Racing Companion v1.1.0

- Lokális autentikáció hozzáadva (regisztráció, bejelentkezés, kijelentkezés)

- Szerepkör-alapú UI megjelenítés (ADMIN / MEMBER)
- Input biztonsági réteg megerősítve (`InputSecurity`)
- Unit és Compose UI tesztek bővítve
