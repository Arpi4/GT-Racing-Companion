# GT-Racing-Companion

Egy motorsport témájú alkalmazás, amely lehetővé teszi pilóták, csapatok, versenyek és felhasználói edzésnaplók kezelését. Az alkalmazás az alap CRUD műveleteket valósítja meg: létrehozás, lekérdezés, módosítás és törlés, valamint a különböző entitások közötti logikai kapcsolatokat kezeli. A cél a felhasználók számára egy vizuálisan áttekinthető, könnyen használható felület biztosítása a GT-versenyek nyomon követéséhez.
